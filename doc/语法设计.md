# Zeta 语言 语法规范

## 1. 词法结构

### 1.1. 标识符

- `utf-8` 编码

- 字母、数字、下划线组成，不能以数字开头。
- 允许 Unicode 字符。

### 1.2. 关键字

```
break       class       const       continue    else        extern
false       fn          for         if          import      let
null        public      private     return      static      super
this        true        var         while
```

### 1.3. 运算符

| 优先级 | 类别      | 运算符                                                  | 结合性               |
| :----- | :-------- | :------------------------------------------------------ | :------------------- |
| 1      | 成员/调用 | `.` `[]` `()`                                           | 左                   |
| 2      | 后缀      | `++` `--`                                               | 左                   |
| 3      | 前缀      | `++` `--` `!` `~` `+` `-`                               | 右                   |
| 4      | 乘除取余  | `*` `/` `%`                                             | 左                   |
| 5      | 加减      | `+` `-`                                                 | 左                   |
| 6      | 移位      | `<<` `>>`                                               | 左                   |
| 7      | 位与      | `&`                                                     | 左                   |
| 8      | 位异或    | `^`                                                     | 左                   |
| 9      | 位或      | `|`                                                     | 左                   |
| 10     | 比较      | `<` `<=` `>` `>=`                                       | 左（不支持链式比较） |
| 11     | 等值      | `==` `!=`                                               | 左                   |
| 12     | 逻辑与    | `&&`                                                    | 左                   |
| 13     | 逻辑或    | `||`                                                    | 左                   |
| 14     | 三元条件  | `? :`                                                   | 右                   |
| 15     | 赋值      | `=` `+=` `-=` `*=` `/=` `%=` `&=` `|=` `^=` `<<=` `>>=` | 右                   |
| 16     | 逗号      | `,`                                                     | 左                   |

### 1.4. 其他符号

- 语句结束符：分号 `;`
- 代码块：花括号 `{ }`
- 注释：`//` 单行，`/* */` 多行

### 1.5. 字面量

| 类型   | 示例                                          |
| ------ | --------------------------------------------- |
| 整数   | `123`, `0xFF`, `0b1010`, `0o755`, `1_000_000` |
| 浮点数 | `3.14`, `1e-4`                                |
| 字符串 | `"hello"`, `"escape \n"`                      |
| 布尔   | `true`, `false`                               |
| 空值   | `null`                                        |
| 数组   | `[1, 2, 3]`                                   |
| 映射   | `{ name: "Alice", age: 25 }`                  |

## 2. 值与数据类型

Zeta 是动态类型语言，变量不绑定类型，值携带类型。

### 2.1. 整数（Integer）

- 任意精度（实现可用 64 位），支持常见字面量。

### 2.2. 浮点数（Float）

- 双精度浮点数。

### 2.3. 字符串（String）

- 支持转义序列。

### 2.4. 布尔（Bool）

- `true` / `false`

### 2.5. 空值（Null）

- `null`，表示无值。

### 2.6. 数组（Array）

- 动态数组，可容纳混合类型。
- 字面量 `[1, "a", true]`
- 索引访问 `arr[0]`
- 属性：`len`
- 方法：`push()`, `pop()`, `clear()`

### 2.7. 映射（Map）

- 键值对集合，键为整数或字符串。
- 字面量 `{ key: value }`
- 访问 `map["key"]`
- 方法：`len`, `has()`, `remove()`

### 2.8. 函数（Function）

- 第一类值，支持闭包。

### 2.9. 类实例（Instance）

- 由类构造创建，包含字段和方法。

## 3. 变量

- 首次赋值即定义变量，未定义变量不可使用。
- 可变变量：`var x = 10;` 或直接 `x = 10;`
- 不可变变量：`let y = 20;`（不能重新赋值）
- 作用域：块级作用域（`{ }` 内有效）。

## 4. 语句

### 4.1. 块

```cpp
{
    statement1;
    statement2;
}
```

### 4.2. 控制流

```cpp
if (cond) { ... } else if (cond2) { ... } else { ... }
while (cond) { ... }
do { ... } while (cond);
```

### 4.3. 循环

```cpp
// C 风格 for
for (var i = 0; i < 10; i++) { ... }

// for-in 遍历容器
for (var item : container) { ... }
```

### 4.4. break / continue

跳出当前循环或继续下一次迭代。

### 4.5. return

```cpp
return;      // 返回 null
return expr; // 返回一个值
```

### 4.6. 变量声明

```cpp
var x = 10;
let y = 20;
```

### 4.7. 函数声明

```cpp
fn add(a, b) {
    return a + b;
}
```

### 4.8. 类声明

```cpp
class MyClass {
    var field = 0;
    fn MyClass(param) { this.field = param; }
    fn method() { ... }
}
```

### 4.9. 常量与枚举

```cpp
const PI = 3.14
enum Color { RED, GREEN, BLUE }
```

### 4.10. 表达式语句

任何表达式后跟分号，如赋值、函数调用等。

## 5. 表达式

### 5.1. 赋值

`=` `+=` `-=` `*=` `/=` `%=` `&=` `|=` `^=` `<<=` `>>=`

### 5.2. 运算符

- 算术：`+` `-` `*` `/` `%`
- 自增自减：`++` `--`
- 位运算：`&` `|` `^` `~` `<<` `>>`
- 逻辑：`&&` `||` `!`
- 比较：`==` `!=` `<` `<=` `>` `>=`
- 三元条件：`? :`
- 成员访问：`.` `[]`
- 函数调用：`()`

## 6. 函数

- 参数个数必须匹配（不支持默认参数或可变参数，未来可扩展）。
- 匿名函数：`fn(x) { return x*x; }`
- 闭包：内层函数可以访问外层函数的局部变量。

## 7. 类

### 7.1. 类声明

```cpp
class ClassName {
    // 字段（默认 public）
    var field = init;
    let CONST = value;
    
    // 构造函数（与类同名）
    fn ClassName(params) { ... }
    
    // 方法
    fn method(params) { ... }
    
    // 静态方法
    static fn staticMethod() { ... }
    
    // 私有成员
    private var privField;
    private fn privMethod() { ... }
}
```

### 7.2. 继承

```cpp
class Derived extends Base {
    fn Derived(params) {
        super(params);   // 调用基类构造
    }
    fn method() override { ... }
}
```

## 8. 模块

- 每个 `.zt` 文件是一个模块。
- 默认顶层成员私有，使用 `public` 导出。
- 导入：
```cpp
import "module.zt";
import "module.zt" as M;
import { f, g } from "module.zt";
```

## 9. 与 C++ 互操作

### 9.1. 外部函数

```cpp
extern fn getDeltaTime();
extern fn setWindowTitle(title);
```

### 9.2. 外部类

```cpp
extern class Sprite {
    fn load(path);
    fn draw();
}
```

## 10. 内置函数

- `print(...)`：输出到控制台。
- `typeof(expr)`：返回类型字符串。
- `assert(cond)`：条件为假时中止。
